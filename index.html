Variables

One of the most important things in any programming language are called variables. You can think of variables as if they were a box with stuff in them. Here is an example of how a variable might be used in a real program:

emails = 10 print("You have:") print(emails) print("email(s)")

This may be a little confusing so let me break it down for you.


First, at the very start of the program, we set the variable like this:

  1. Type the variable's name. This MUST start with either an upper/lowercase character or an underscore (_)
  2. Type a space.
  3. Type an equals sign (=). This assigns the variable with its property. You can think of it like labeling a box.
  4. Type your variable. This can be a number, like in the example above, a string of text (remember to enclose that string of text in either single ('') or double ("") quotes), or another variable.

What does the print() function do? This function simply writes a line of text onto the screen. So, here is what all of those print() functions do:

  1. The first print() writes "You have:" onto the screen
  2. Then, the next print() writes our variable onto the screen.
  3. Finally, the last print() writes "email(s)" onto the screen.

Loops (for, while)

Loops are an incredibly important part of all code. They are used primarily to repeat sections of code multiple times without writing it multiple times. There are two different types of loops, which are both shown below.

For loops:

for x in range(2, 6):      print(x)
This code will output:
2 3 4 5

If you couldn't tell, for loops will take a variable, in this case x, and repeat it through a function or string, which is range(2,6) here. Every time the variable (x) increments, all the code indented in the loop will run.

While loops:

i = 1 while i < 6:      print(i)      if i == 3:           print("i equals 3")      i += 1
This code will output:
1 2 3 i equals 3 4 5

While loops run the code indented below them until the condition specified (i < 6 in this case) is false. This can be used to create an infinite loop (While True:), though this can be a stress to your computer and is generally considered bad practice.

Conditionals (if-else statements)

Conditionals tell your computer whether or not to do something. You have already seen an example of this above with the if i == 3 in the while loop. The most common types of conditionals are if and if-else statements.

a = 200 b = 33 if b > a:      print("b is greater than a") elif a == b:      print("a and b are equal") else:      print("a is greater than b")

This might look complicated, but in reality it's pretty simple. First off we define a as 200 and b as 33. Then we get to the first if statement (if b > a), which is going to run the code indented below if the condition is true, which in this case it won't because b isn't greater than a. Now the next statement, the elif is a combined else and if statement. This means that it will run if the condition in the if statement (b > a) isn't true, AND if the condition in the elif (a == b) is true. Keep in mind that in python you use double equals (==) for an if statement, since a single equals sign will change one value to another, i.e. a = 10 changes the value of a to 10, while a == 10 returns true if a is equal to 10. Finally, the last statement, the else: line. This will run if the if, or in this case elif, returns false.

So let's see this in action! With a equal to 200 and b to 33, we get the following result:

a is greater than b

Let's work through this. First the computer goes to the if statement and checks if b > a, which it isn't, so it move on to the elif. this also returns false becuase 200 does not equal 33. this leaves only the else statement, which prints out that a is greater than b. Notice that we didn't check if a was actually greater than b, because we know it has to be if both b > a and a == b aren't true. Now let's try with different numbers, say a = 2 and b = 10.

b is greater than a

Here we still start with the first if, but this time it returns true, so we run that first print statement, and since that if returned true, the computer doesn't run the elif or the else. Finally, if a and b were both the same number, say 20, what would happen?

a and b are equal

You probably expected that, but just to make sure you know, here's what happened. We starrt with the if again, but since it's false we move on to the elif. We treat elifs much the same as normal ifs, so we check its condition and it's true, so we print that a equals b. And again, since the elif was true, we do not run the else statement.

Functions

Functions are like a reusable bag or backpack. They contain code that can be resued any time you want. This is incredebly useful if you have a lot of code in different places that are similar. You can even add variables into functions for different outputs, which you can see below.

def name_function(fname, lname):      print(fname + " " + lname) name_function("John", "Doe")
Outputs:
John Doe

This code probably seems confusing, but let's break it down.

  1. def name_function(fname, lname):. This is the line that defines the function. We always start this with def, then the name of the function and some parentheses. The fname and lname are the function variables, and are always put in those parentheses.
  2. print(fname + " " + lname). This line is the content of our function, and what will be doen when we call our function. In this case we print out what we define for the fname, a space, and lname.
  3. name_function("John", "Doe"). This is where we call our function, and where the code actually runs. First we write the name of the function and then we put the values we want for fname and lname in the parentheses, which in this case is John for fname and Doe for lname. This will run our function with these variables, which is the print statement from before.

Lists

And finally, the last part of this lesson, lists. These are pretty much exaclty what they sound like, just a list of values stored like a variable. These values can be strings (letters), numbers, variables, or even other lists. Let's look at an example.

fruits = ["apple", "banana", "cherry"] print(fruits[0]) print(fruits[1]) print(fruits[2])
Outputs:
apple banana cherry

First, we create a variable called fruits and assign a list to it. A list is just a collection of objects. We can print this list out using the print() function like in the example.

An important thing to know about Python is that, like some other languages, the first value in a list has the value of 0, the second object has the value of 1, the third having the value of 2, and so on. That is why we start by printing the 0th object in the list first instead of the 1st object; because the 1st object is the second object.
  1. The first print writes the object in the 0 position; or the first object in the list, which is "apple".
  2. The second print writes the object in the 1 position; or the second object in the list, which is "banana".
  3. The last print writes the object in the 2 position; or the third object in the list, which is "cherry".

Variables

One of the most important things in any programming language are called variables. You can think of variables as if they were a box with stuff in them. Here is an example of how a variable might be used in a real program:

emails = 10 print("You have:") print(emails) print("email(s)")

This may be a little confusing so let me break it down for you.


First, at the very start of the program, we set the variable like this:

  1. Type the variable's name. This MUST start with either an upper/lowercase character or an underscore (_)
  2. Type a space.
  3. Type an equals sign (=). This assigns the variable with its property. You can think of it like labeling a box.
  4. Type your variable. This can be a number, like in the example above, a string of text (remember to enclose that string of text in either single ('') or double ("") quotes), or another variable.

What does the print() function do? This function simply writes a line of text onto the screen. So, here is what all of those print() functions do:

  1. The first print() writes "You have:" onto the screen
  2. Then, the next print() writes our variable onto the screen.
  3. Finally, the last print() writes "email(s)" onto the screen.

Loops (for, while)

Loops are an incredibly important part of all code. They are used primarily to repeat sections of code multiple times without writing it multiple times. There are two different types of loops, which are both shown below.

For loops:

for x in range(2, 6):      print(x)
This code will output:
2 3 4 5

If you couldn't tell, for loops will take a variable, in this case x, and repeat it through a function or string, which is range(2,6) here. Every time the variable (x) increments, all the code indented in the loop will run.

For loops:

i = 1 while i < 6:      print(i)      if i == 3:           print("i equals 3")      i += 1
This code will output:
1 2 3 i equals 3 4 5

While loops run the code indented below them until the condition specified (i < 6 in this case) is false. this can be used to create an infinite loop (While True:), though this can be a stress to your computer and is generally considered bad practice.
